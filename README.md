# Премиум Поздравление с Днём Рождения

Интерактивный cinematic веб-проект-поздравление для Ак Джамолидина.

## Как запустить

Откройте файл `index.html` в браузере. Проект не требует сервера.

## Как изменить имя

В файле `index.html` найдите текст `Ак Джамолидин` и замените на нужное имя.

## Как изменить текст поздравления

Текст поздравления находится в `index.html` внутри блока с классом `letter-body` (сцена 4).

## Как добавить фотографии

Добавьте изображения в папку `assets/images/` и вставьте тег `<img>` в нужное место в `index.html`.

## Как добавить музыку

Поместите аудиофайл (MP3) в `assets/audio/birthday.mp3`. Музыка будет доступна через кнопку звука в правом верхнем углу.

## Сцены проекта

1. **Интро** — загадочный экран с текстом
2. **Конверт** — 3D-конверт с hover-эффектом
3. **Открытие** — анимация вскрытия конверта
4. **Письмо** — постепенное появление текста поздравления
5. **Финал** — праздничная анимация с конфетти и фейерверками

## Технологии

- HTML5
- CSS3 (3D transforms, glassmorphism, animations)
- Vanilla JavaScript
- Canvas API (конфетти, фейерверки)
